\begin{frame}
\frametitle{Where are we, with regard to track-\\ based/hardware comparisons?}
\framesubtitle{This is my understanding:}
\small
\begin{itemize}
\item Track-based minus hardware differences show a relative twist (global $\Delta r\phi$ vs $z$) and a compression (global $\Delta z$ vs $z$) of about the same magnitude, roughly the same in all sectors, all stations
\item Present at the level of raw residuals
\item Not in segment-to-segment residuals, but not expected
\item Several chambers appear to be displaced in the global $z$ direction in the hardware geometry (discontinuities in residuals)
\item Link system and inclinometers both find the apparent twist to be 1 $\pm$ small~mm between ends of the barrel (track-based sees 4~mm).  These are independent of each other and independent of the barrel measurement (track-based is the odd one out)
\item They depend on alignment ring stability since 2009, but there are some good indications of that
\item This is the third week: are we any closer?  Is there anything we can do to add new information?
\end{itemize}
\label{numpages}
\end{frame}
